# bilibili 状态码记录

| 状态码 | 备注                                                         |
| ------ | ------------------------------------------------------------ |
| 0      | 请求执行成功                                                 |
| -403   | 访问权限不足，原因是 http 请求头中的 user-agent、referer 和 cookie 存在问题 |
| -10500 | bilibili 服务器会回复“处理失败”，原因暂时还不知道，可能是 socket 的保活时间快到了 |
